- Closed a declassify-guard bypass: the `trusted-filer-mutate` matcher blocking `gh issue edit <N> --remove-label directive` only matched the space-separated form, so the `=`-separated form (`--remove-label=directive`, valid `gh` syntax) silently dropped the `directive` label and de-classified a Directive without the required human confirm. The matcher now accepts both separator forms (optionally quoted) and a comma-joined value list where `directive` is any element (`--remove-label other,directive`, which gh honors), with a word-boundary tail so a longer label like `directive-foo` does not over-match.
